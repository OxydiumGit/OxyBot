Welcome to My Discord Bot Documentation

This documentation will guide you through the commands and features of My Discord Bot.

Command Syntax

Commands in this bot follow this general syntax: !command [arguments]. Also the bot supports slash (/) commands, which is recommended.

General Commands

  • !help - Displays help information.
  • !ping - If the bot is alive, also responds with the latency of the bot.
  • !botinfo - Shows info about the bot like owner of the bot, amount of servers the bot is in, the python version the bot is running on, and the version of the bot.
  • !serverinfo - Shows info about the server like the server name, guild id, member count, and text/voice channel count.
  • !invite - Sends the user a link to invite the bot to a server.
  • !server - Sends the user a link to the support server of the bot.
  • !8ball [question] - responds with a response from an 8ball
  • !bitcoin - responds with the current price of bitcoin
  • !userstats [user] - Responds with info about a user like thwir account age, messages sent in the server, date they joined the server, and how much bot currecny they have. Optionally, you can run the command without specifing a user to get your own stats.
  • !leaderboard - Responds with a leaderboard ranking users by message count in the server.

Moderation Commands

  • !kick [user] - Kicks a user from the server.
  • !nick [user] [nickname] - Changes a users nickname.
  • !ban [user] [reason] - Bans a user from the server, reason is optional.
